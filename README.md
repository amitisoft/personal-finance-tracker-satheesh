# Personal Finance Tracker

Personal Finance Tracker is a web application for managing day-to-day finances in one place. It helps users track accounts, transactions, budgets, goals, recurring entries, reports, and shared financial access through a frontend and backend application managed in this repository.

This repository contains:

- `personal-finance-tracker-frontend`
- `personal-finance-tracker-backend`
